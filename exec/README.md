## inbound 포트

webRTC와 연결할 포트를 열어줍니다
```
sudo iptables -I INPUT 1 -p tcp --dport 3478 -j ACCEPT
sudo iptables -I INPUT 1 -p udp --dport 3478 -j ACCEPT
sudo iptables -I INPUT 1 -p udp --dport 49152:65535 -j ACCEPT

```

## 프로그램 실행

깃에서 파일을 클론합니다.
```
git clone https://lab.ssafy.com/s06-webmobile1-sub2/S06P12A107.git
```

node_modules를 받습니다.
```
cd S06P12A107
cd frontend
npm install
```


실행
```
cd ../backend
gradle bootrun
```
프록시 패스를 설정하지 않는 경우 i6a107.p.ssafy.io:8443으로 접속하실 수 있습니다.
